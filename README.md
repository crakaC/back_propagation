###なにこれ
誤差逆伝播法（Back Propagation）のアルゴリズムです。

###How to use
1. cloneするか、zipで落として適当なディレクトリに展開してください。
2. src内ででmakeすれば実行ファイルができる（と思う）ので、それをぽちっとやってください。

    make
    ./backpropagation
    1:訓練データ入力
    2:学習
    3:試してみる
    4:一括パラメータ調整
    5:パラメータ調整
    6:訓練データ作成
    0:終了
    >
